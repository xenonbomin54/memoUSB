## 프로젝트 소개
**memoUSB**는 프레임워크 없이 **Vanilla JS**와 **CSS 애니메이션**만으로 구현된 가볍고 세련된 메모 도구입니다. 복잡한 설정 없이 브라우저에서 즉시 메모를 작성하고 안전하게 보관할 수 있습니다.

<br/>

## 핵심 기능 및 특징
*   **Smooth Animation**: CSS를 활용한 부드러운 전환 효과로 시각적 즐거움을 제공합니다.
*   **Data Persistence**: `LocalStorage`를 사용하여 브라우저를 새로고침하거나 닫아도 데이터가 유지됩니다.
*   **Pure Web Tech**: 별도의 라이브러리(React, Vue 등) 없이 순수 웹 표준 기술로만 제작되었습니다.
*   **Responsive Design**: 다양한 화면 크기에서도 깔끔하게 보이는 반응형 UI를 지원합니다.

<br/>
